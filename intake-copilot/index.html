How the screen works

        
Fields arrive with extraction confidence (bars). Policy rules run on every document: duplicate detection, PO requirements by amount, vendor-master and W-9 checks, contract-rate comparison, and approval-tier thresholds. Verdicts: auto-post (no touch), verify (confirm one field), exception (approver decision with evidence). This demo is rule-based and fully client-side; the production version is an agentic extraction pipeline with an evaluation harness, model-confidence routing, and posting into the systems of record.

      

    

  


  

    
How this hardens for production
